# Business KPI Dashboard

An interactive Business KPI Dashboard built using Python, SQL, Pandas, and Plotly Dash to track and analyze key business metrics.

## Features
- Total Revenue KPI
- Revenue by Region Analysis
- Revenue Trend Visualization
- Automated Data Loading
- SQL-based Data Processing
- Interactive Dashboard UI

## Tech Stack
- Python
- Pandas
- SQLite (SQL)
- Plotly
- Dash

## Project Structure
business-kpi-dashboard/
│
├── data/            # Dataset
├── sql/             # SQL queries
├── app/             # Dashboard code
├── requirements.txt
├── README.md
└── .gitignore

## Run Locally
pip install -r requirements.txt
python app/dashboard.py
